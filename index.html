QIWI OSS License

MIT compatible and corresponding to the Civil Code of the Russian Federation.

  • License in English (en)
  • Лицензия на русском (ru)

Install

 
yarn add @qiwi/license
npm i @qiwi/license

Usage

Through CLI

qiwilicense name='LICENSE' dir=/some/path lang=ru year=2019

TS/JS

import {generate, render} from '@qiwi/license'

const text = render({
  lang: 'ru',
  year: '2010-2019'
})
// ... do something

// Or render and write to a file:
generate({
  name: 'LIC',
  dir: resolve(__dirname, '../foo/bar'),
  lang: 'en',
  year: '2019'
})

Option Description Default
